# 📊 Bank Marketing Campaign Analysis

A professional data analytics project using **R** to analyze and predict customer behavior for a retail bank's marketing campaign.

## 🎯 Project Objectives
- Analyze customer demographics (Job, Marital Status, Balance).
- Perform statistical hypothesis testing (Chi-Square).
- Build a **Logistic Regression** model to predict subscription success.

## 📂 Repository Structure
- `Data_Cleaning.R`: Pre-processing and handling semicolon-separated values.
- `Exploratory_Analysis.R`: Visualizations and statistical testing.
- `Predictive_Model.R`: Logistic regression modeling and coefficient analysis.
- `Analysis_Report.Rmd`: A comprehensive R Markdown report.

## 💡 Key Business Insights
1. **Top Performers:** Students and retirees are significantly more likely to subscribe.
2. **Call Duration:** Longer conversations are highly correlated with successful conversions.
3. **Financial Profile:** Higher yearly balances positively influence the decision to open a term deposit.

## 🚀 How to Run
1. Clone this repository.
2. Ensure `bank-full.csv` is in the root directory.
3. Run the scripts in order: `Data_Cleaning.R` -> `Exploratory_Analysis.R` -> `Predictive_Model.R`.
